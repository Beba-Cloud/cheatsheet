CSS Selectors and Specificity

Selectors

Selector Description Example
Universal It selects all the elements of any type * {font-family: Verdana;}
Type It matches the type of element in the HTML document p {color: green;}
Class It selects HTML elements with a specific class name .brand { }
ID It uses the ID attribute of an HTML element to select a specific element #large-title { }
Attribute It selects HTML elements based on the presence of a specific attribute or the value of that attribute [href] {color: magenta;}
Pseudo-Class It adds style to elements based on their state p:hover {background-color: lime;}

Specificity

Selector Type Priority
ID Highest priority, identified by the unique id attribute of an element
Classes and Pseudo-classes Second highest priority, targeted using class names
Type Lowest priority, applies to HTML elements
